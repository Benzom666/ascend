# ✨ Le Society Features

Complete overview of platform features.

## For Female Users (Date Creators)

### Profile Management
- Multi-step registration with photo upload
- Profile verification system (photo + admin approval)
- Detailed profile customization (bio, preferences, photos)
- Privacy controls

### Date Management
- Create premium date experiences
- Set location, time, duration
- Specify earning expectations
- Upload date photos
- Manage active/past dates
- Edit or delete dates

### Interaction Control
- Token-gated messaging (men must purchase tokens)
- Accept or reject chat requests
- Block unwanted users
- View interest notifications (standard & super-interest)

### Safety Features
- Admin moderation of profiles and dates
- Report system for inappropriate behavior
- Verification badges
- Privacy settings

---

## For Male Users (Date Seekers)

### Discovery
- Browse verified female profiles
- Filter by location, age, interests
- View active date listings
- See profile verification status

### Interest System
- Express interest (free)
- Send super-interest (5 tokens)
- View who you've shown interest to
- Save favorites

### Messaging
- Purchase token packages ($10-50)
- Initiate chats (10 tokens)
- Send messages (2 tokens per message)
- Real-time chat with history
- View chat status

### Profile
- Create detailed profile
- Upload photos
- Set preferences
- View token balance
- Manage subscriptions
