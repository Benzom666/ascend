# ElPay Payment Integration API

A guide for merchants who want to accept payments programmatically.

## What you can do

- **Create a payment** — generate a hosted payment page and redirect your customer there
- **Query a payment** — poll the status of any payment by ID
- **List assets** — discover which currencies your store can accept
- **Receive webhooks** — get notified instantly when a payment status changes

All endpoints live under:

```
https://api.elpay.online/int/
```
